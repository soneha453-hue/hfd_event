# HFD-EVENT — Premium Event Decoration Services

**Live demo:** [hfd-event.web.app](https://hfd-event.web.app)

A customer-facing booking platform for event decoration services (weddings, birthdays, engagements, and other occasions), with a CRUD admin panel for the business owner and a flexible payment system designed around a real customer pain point.

## Built using AI-assisted development

This project was built end-to-end using AI coding tools. My role was to identify a real-world problem, define the requirements, prompt and iterate with AI to build the solution, and test/validate the final product. This project is meant to demonstrate the ability to use AI tools to design, build, and ship a real, working product — not just write code from scratch.

## The problem

Event decoration bookings are often shared expenses (e.g. a family or group splitting the cost of wedding decor), but not everyone can pay their share on time. Traditional booking platforms expect full payment upfront, which doesn't reflect this real-world constraint.

## The solution

- **Flexible/partial payments** — customers can pay in installments, and if a payment is delayed, they can send a request to the owner to extend the due date — instead of being penalized or losing the booking.
- **Customer-facing catalog** — browse decoration packages by category, filter by price range, and sort by newest, price, or rating.
- **Customer reviews** — social proof for past clients' experiences.
- **Owner admin panel** — full CRUD (Create, Read, Update, Delete) for managing decoration categories, items, and pricing.

## Features

- Category-based browsing of decoration packages
- Price filters (Under ₹10,000 / ₹10,000–₹25,000 / ₹25,000+)
- Sorting (Newest, Price: Low to High, Price: High to Low, Highest Rated)
- Customer review display
- Owner-only CRUD panel for managing listings
- Flexible payment workflow with owner-approved date extensions
- Contact section for direct booking inquiries

## Tech stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend / Database:** Firebase (Hosting, Firestore)

## What this project demonstrates

- Translating a real-world constraint (group payments) into a concrete product feature
- Defining requirements and iterating through prompts to reach a working solution with AI tools
- Testing and validating CRUD operations and payment workflows end-to-end
- Deploying a live, production-accessible product on Firebase Hosting
